# Claude Voice Assistant for Home Assistant

A custom integration that adds Claude AI capabilities to Home Assistant voice commands.

## Features
- Custom wake words with Claude AI responses
- ElevenLabs voice synthesis
- Full Home Assistant device control
- Natural language understanding

## Installation
1. Add this repository to HACS
2. Install the integration
3. Configure with your API keys
4. Set up your voice assistants

## Configuration
After installation, you'll need:
- Anthropic API key
- ElevenLabs API key

## Version
Current version: 1.0.0
